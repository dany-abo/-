# 🎓 منصة كورسات+ | Courses Plus Platform

منصة تعليمية إلكترونية متكاملة تتيح للطلاب تصفح الكورسات، التسجيل فيها، ومتابعة التقدم من لوحة تحكم شخصية. المشروع مبني بالكامل باستخدام **HTML5** و **Tailwind CSS** و **Vanilla JavaScript** بدون أي Frameworks.

---

## ✨ التحديثات الجديدة

### 🆕 الصفحات الجديدة (أصبح المجموع 6 صفحات)
- ✅ **course-details.html** - صفحة تفاصيل الكورس الشاملة
- ✅ **login.html** - صفحة تسجيل الدخول الأنيقة
- ✅ **404.html** - صفحة خطأ 404 احترافية مع تأثير Glitch **[جديد]**

- قسم "لماذا نحن" مع 4 مميزات
- قسم آراء الطلاب (Testimonials)
- قسم CTA يشجع على التسجيل

### 🎨 تحسينات التصميم
- ✅ **Favicon** مخصص لجميع الصفحات (أيقونة 🎓)
- ✅ **404 Page** احترافية مع تأثيرات متقدمة
- ✅ **Custom Cursor** متطور على جميع الصفحات

## كيفية تشغيل المشروع

### الطريقة 1: فتح مباشر
1. افتح مجلد المشروع
2. انقر مرتين على `index.html` لفتحه في المتصفح

### الطريقة 2: Live Server (موصى بها)
1. افتح المجلد في **VS Code**
2. ثبّت إضافة **Live Server**
3. انقر بزر الماوس الأيمن على `index.html` → **Open with Live Server**

### الطريقة 3: سيرفر محلي
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
ثم افتح `http://localhost:8080` في المتصفح.

## المميزات

- **6 صفحات كاملة** — الرئيسية، الكورسات، تفاصيل الكورس، لوحة التحكم، تسجيل الدخول، صفحة 404
- **Favicon مخصص** — أيقونة 🎓 احترافية تظهر في Tab المتصفح
- **Dark Mode** — وضع ليلي/نهاري مع حفظ التفضيل في LocalStorage
- **LocalStorage** — حفظ الكورسات المسجلة، بيانات المستخدم، وتقدم الطالب
- **Animations** — Typing Effect، Animated Counters، Skeleton Loader، Page Transitions، Custom Cursor، Accordion، Glitch Effect
- **Responsive Design** — متجاوب 100% مع جميع الأجهزة (موبايل، تابلت، ديسكتوب)
- **البحث والفلترة** — بحث فوري وتصفية حسب التصنيف في صفحة الكورسات
- **Toast Notifications** — إشعارات منبثقة عند التسجيل، الإزالة، وتغيير الوضع
- **Loading Screen** — شاشة تحميل عند أول زيارة للموقع
- **Form Validation** — التحقق من صحة النماذج (نشرة بريدية، تسجيل دخول)
- **Authentication** — نظام تسجيل دخول مع حفظ بيانات المستخدم
- **Course Details** — صفحة تفاصيل شاملة لكل كورس مع Accordion للمحتوى
- **404 Page** — صفحة خطأ احترافية مع تأثير Glitch وتصميم متحرك
