# منصة كورسات+ | Courses Plus Platform

منصة تعليمية إلكترونية متكاملة تتيح للطلاب تصفح الكورسات، التسجيل فيها، ومتابعة التقدم من لوحة تحكم شخصية. المشروع مبني بالكامل باستخدام **HTML5** و **Tailwind CSS** و **Vanilla JavaScript** بدون أي Frameworks.

---

## التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **HTML5** | هيكل الصفحات (الرئيسية، الكورسات، لوحة التحكم) |
| **Tailwind CSS** | التصميم والتنسيق عبر CDN |
| **Vanilla JavaScript (ES6+)** | المنطق الديناميكي والتفاعلات |
| **LocalStorage** | حفظ الكورسات المسجلة وتفضيل الوضع الليلي |
| **AOS** | حركات الظهور عند التمرير |
| **GSAP** | حركات متقدمة (Typing، Counters، Grid Animations) |
| **Font Awesome** | الأيقونات |
| **Google Fonts** | خطوط Cairo و Tajawal و Poppins |

---

## المميزات

- **Dark Mode** — وضع ليلي/نهاري مع حفظ التفضيل في LocalStorage
- **LocalStorage** — حفظ الكورسات المسجلة وتقدم الطالب
- **Animations** — Typing Effect، Animated Counters، Skeleton Loader، Page Transitions، Custom Cursor
- **Responsive Design** — متجاوب بالكامل مع الموبايل والتابلت وسطح المكتب
- **البحث والفلترة** — بحث فوري وتصفية حسب التصنيف في صفحة الكورسات
- **Toast Notifications** — إشعارات منبثقة عند التسجيل، الإزالة، وتغيير الوضع
- **Loading Screen** — شاشة تحميل عند أول زيارة للموقع
- **Form Validation** — التحقق من صحة نموذج الاشتراك في النشرة البريدية

## كيفية تشغيل المشروع

### الطريقة 1: فتح مباشر
1. افتح مجلد المشروع
2. انقر مرتين على `index.html` لفتحه في المتصفح

### الطريقة 2: Live Server (موصى بها)
1. افتح المجلد في **VS Code**
2. ثبّت إضافة **Live Server**
3. انقر بزر الماوس الأيمن على `index.html` → **Open with Live Server**

### الطريقة 3: سيرفر محلي
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
ثم افتح `http://localhost:8080` في المتصفح.
